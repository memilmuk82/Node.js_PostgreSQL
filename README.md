구현 코드, 런타임 버전, 의존성과 데이터베이스 구성이 정해져 있지 않아 제공할 수 있는 설치·실행 명령이 없습니다. 구현이 추가된 뒤 실제 기술과 설정에 맞춰 이 절을 작성해야 합니다.

구현을 시작하기 전에 다음 원칙을 적용해야 합니다.

- 비밀번호, 연결 URI, 인증서와 그 밖의 비밀값을 코드·문서·Git 기록에 저장하지 않습니다.
